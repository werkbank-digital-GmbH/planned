Legende / Visual Language

A

Active Chip

Represents the source of the interaction. Solid fill indicates a confirmed or currently held object.

G

Ghost Chip

Indicates the potential target range. Opacity and dashed borders signify a temporary "preview" state.

Connection Line

Visually links the start and end points of a range selection, reinforcing continuity.
